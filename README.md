# EEng

영어 학습 웹 애플리케이션

## ✏️ 역할
- PM: 구예림
- 팀원: 김서영

## 🗓️ 개발 일정

2024.03 ~ 2024.06 [4개월]

## 📌 주요 기능

### 김서영
> - 회원가입 및 로그인/로그아웃
>     - 로그인 미완료시 다른 기능 사용 불가
> - 영어책 번역 및 음성 출력
>     - 문장 별 한국어 번역 지원
>     - TTS 기능으로 음성 출력

### 구예림
> - 레벨 테스트
>     - 3가지 유형의 테스트 문제
>     - 측정된 레벨에 맞는 학습 단계 추천
> - 단어장을 통한 단어 공부
>     - 단어의 뜻 표기를 사용자 개인이 설정하여 맞춤 학습
>     - TTS 기능으로 발음 연습
